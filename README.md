# Images Annotator

Images annotator is simple browser-based tool ([try it here](https://edinburgh-genome-foundry.github.io/ImagesAnnotator)) to annotate a series of pictures.

It was developed by Zulko at the Edinburgh Genome Foundry as a useful toy project and is released on Github under the MIT licence (Â¢ Edinburgh Genome Foundry, 2017).

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
